Arduinoで温度・照度センサーの値を取得し、LoRa 通信を通じて Raspberry Pi に送信し、Linux 環境で構築した Python、サーバーで受信・保存する IoT データロギングシステムを構築しています。
Arduino 側ではセンサー制御・SD保存・通信処理を行い、デバッグのためにシリアル出力やLCDディスプレイも実装しました。

Raspberry Pi 側では Python によるデータ処理スクリプトを作成し、一連の動作を安定して行えるよう環境構築を行いました。

今後Java を用いた Android アプリ開発に取り組み、最終的には、Raspberry Pi に蓄積されたデータを Android アプリ上で確認できる仕組みを実装しこちらに追加していきます。

これらを通じて、設計・電子回路の組み立て・実装・通信処理・デバッグまで、一連の開発工程を実践形式で経験しました。
